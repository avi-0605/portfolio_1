# 🚀 Run Locally

Clone the project

```bash
git clone https://github.com/avi-0605/portfolio_1.git
```

Go to the project directory

```bash
cd portfolio_1
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
